go run ./src/cmd/main.go

json dapat diakses di:
http://localhost:8080/graph-data

http://localhost:8080/api/recipes/{nama_item_yang_mau_dicari_resepnya}?algorithm=...

centang pilihan pretty-print di kiri atas

kalo ada error coba:
go mod tidy
